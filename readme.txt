== Description ==

= Overview =

This plugin adds plenty of premium functionalities to Alpus theme, such as elementor page builder elements, header builder elements, and theme options to extend the functionality of that theme.

* Elementor Page builder support (18 Exclusive Elements)
* WooCommerce plugin support (8 Exclusive Elements)
* Custom CSS and Javascript fields
* Header builder support (Side header, Transparent header)
* Mega Menu
* Menu Skins
* Adding widget area
* Fully Responsive

== Installation ==

= Minimum Requirements =

* WordPress 4.6 or greater
* PHP version 5.3.0 or greater
* MySQL version 5.0 or greater


**This plugin works the best in [Alpus Theme](https://alpustheme.com/)**


= Automatic installation (easiest way) =

To do an automatic install of Alpus Core, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Alpus Core" and click Search Plugins. Once you have found it you can install it by simply clicking "Install Now".


= Manual installation =

**Uploading in WordPress Dashboard**

1. Download `alpus-core.zip`
2. Navigate to the 'Add New' in the plugins dashboard
3. Navigate to the 'Upload' area
4. Select `alpus-core.zip` from your computer
5. Click 'Install Now'
6. Activate the plugin in the Plugin dashboard

**Using FTP**

1. Download `alpus-core.zip`
2. Extract the `alpus-core` directory to your computer
3. Upload the `alpus-core` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard

The WordPress codex contains [instructions on how to install a WordPress plugin](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
